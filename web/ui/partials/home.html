Team Manner

An OA software dedicate to all team/group working trace.

TM is mainly using below popular techs:

  • Coding language: JAVA + Javascript
  • Source Control: GitHub (GitHub )
  • Project Management: Maven (Maven)
  • Data Base: Mongo DB (Mongo)
  • CI build: Jenkins
  • Mid-ware layer: Tomcat 7.0 +(Dev)/Glass fish 3.0+(ITG)
  • Front side: Angularjs + bootstrap
  • Backend side: SpringMVC – RESTful web service  (you can learn Restful related knowledge from Jersey )
